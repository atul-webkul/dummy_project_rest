Mobikul Marketplace converts your marketplace store for WooCommerce into a mobile app Android/IOS.

== Installation ==

1. Upload the `wk-mobikul-multivendor-api` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin using the 'Mobikul Marketplace' menu.
